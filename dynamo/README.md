# DynamoDB Local 管理用

* [AWSドキュメント](https://docs.aws.amazon.com/ja_jp/amazondynamodb/latest/developerguide/DynamoDBLocal.UsageNotes.html)

* [ローカルコンソール (http://localhost:8000/shell/)](http://localhost:8000/shell/)

起動
```shell
npm run db_run
```

テーブル作成、一覧
```shell
npm run make_table
npm run list_table
```
